❓ How does auto-save to a file work?

Normally your log lives inside the browser, where clearing browsing data can delete it. Auto-save keeps a live copy in a real file on your computer — one the browser can never delete. Every change you make is written to it automatically.

  1. Click 🔗 Turn on auto-save to a file above.
  2. A save window opens. Choose where to keep the file — your Documents folder is perfect. (Pro tip: picking a OneDrive or Google Drive folder gives you free cloud backup too.)
  3. Click Save. That's it — look for the green ✓ "Auto-saving to T1D-log.json".
  4. When you reopen the app after restarting the browser, it may ask you to hit Reconnect once — that's the browser double-checking permission, not a problem.

If disaster strikes (browser data cleared, log looks empty): open the app, click Reconnect, and it will offer to bring everything back from your file.

Works in Chrome and Edge on computers. On phones, iPads, Safari, and Firefox this feature isn't available — use Save backup file there instead.

🛡️ Tips: stop your browser from deleting the log
Your log is stored by your browser, so the browser's cleanup tools can delete it. Four habits keep it safe:
  • Clearing browsing data? Uncheck "Cookies and other site data" before you clear — or pick "Last hour" instead of "All time". History and cache can go; your log stays.
  • Chrome / Edge / Chromebook: in Settings → Privacy → Site settings → On-device site data, add this site under "Sites that can always save data". That protects the log even if your browser is set to clean up when it closes.
  • iPhone / iPad: use Share → Add to Home Screen. Installed this way, the app gets its own separate storage that Safari's "Clear History and Website Data" doesn't touch.
  • Never use this app in an Incognito or Private window — everything logged there vanishes when the window closes.

And no matter what: the backup file is the only copy your browser can't delete. That's why Poddy nags you about it. 😉
